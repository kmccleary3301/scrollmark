# Store Sync Artifact

`scrollmark.user.js` is the store-safe Scrollmark userscript intended for Greasy Fork/OpenUserJS sync.

Use this raw URL when configuring marketplace sync:

```text
https://raw.githubusercontent.com/kmccleary3301/scrollmark/main/store/scrollmark.user.js
```

Do not use local e2e artifacts for stores. The canonical GitHub direct-install artifact remains available from GitHub Releases as `scrollmark.user.js`.
